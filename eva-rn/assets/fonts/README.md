# Font Files Required

## Plus Jakarta Sans (Free - Google Fonts)
Download from: https://fonts.google.com/specimen/Plus+Jakarta+Sans
Required files:
- PlusJakartaSans-Regular.ttf (weight 400)
- PlusJakartaSans-Medium.ttf (weight 500)
- PlusJakartaSans-SemiBold.ttf (weight 600)
- PlusJakartaSans-Bold.ttf (weight 700)
- PlusJakartaSans-ExtraBold.ttf (weight 800)

## Gilroy (Premium - Radomir Tinkov)
Purchase license from: https://www.myfonts.com/fonts/radomir-tinkov/gilroy/
Required files:
- Gilroy-Regular.ttf (weight 400)
- Gilroy-Medium.ttf (weight 500)
- Gilroy-SemiBold.ttf (weight 600)
- Gilroy-Bold.ttf (weight 700)
- Gilroy-ExtraBold.ttf (weight 800)

Note: Both iOS and Android will automatically pick up fonts placed in this directory when referenced in app.json.
